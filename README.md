# ⚡ Meter Project

🔗 Live Demo: https://tejasmali407.github.io/Meter-Project/

---

## 📌 Project Overview
Meter Project is a simple web application that helps users calculate electricity consumption and estimate their bill based on meter readings.

Users enter previous and current readings, and the system instantly calculates units consumed and the estimated bill amount.

⚠️ This is NOT an official electricity bill. For educational and demonstration purposes only.
---

## 🚀 Features
- Enter previous meter reading
- Enter current meter reading
- Automatic unit calculation
- Instant bill estimation
- Simple and user-friendly UI
- Responsive design (mobile + desktop)
